Git y Github

¿Qué es Git?

Git es un software de control de cambios en diferentes versiones de un código. Está disponible con acceso SSH en todos los planes de alojamiento web de Linux.

Git permite realizar un seguimiento de los cambios en archivos y, si es necesario, restaurar las versiones anteriores. El uso de Git permite a varias personas trabajar simultáneamente en el mismo código sin tener que sobrescribir los cambios del otro.

Además, Git permite la creación de múltiples alternativas. Estas versiones se pueden utilizar para probar nuevos plugins y estructuras de sitios web o para el desarrollo independiente de varios scripts PHP.

¿Qué es Github?

GitHub es una plataforma donde puedes almacenar, compartir y trabajar junto con otros usuarios para escribir código.

Almacenar tu código en un "repositorio" en GitHub te permite:

  • Presentar o compartir el trabajo.
  • Seguir y administrar los cambios en el código a lo largo del tiempo.
  • Dejar que otros usuarios revisen el código y realicen sugerencias para mejorarlo.
  • Colaborar en un proyecto compartido, sin preocuparse de que los cambios afectarán al trabajo de los colaboradores antes de que esté listo para integrarlos.

Comandos básicos de Git

    git add
    Agrega los cambios del directorio al área de trabajo de ensayo.

    git commit
    Confirma los cambios realizados con el comando git add.

    git push
    Publica la contribución confirmada mediante git commit.
